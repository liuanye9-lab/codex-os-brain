# ACOB Agentic Coding Skill

Use this skill shape when a Codex environment supports local skills.

Core behavior:

- run an agentic preflight before complex tasks
- keep context bounded
- dispatch sub-agents only when ROI is positive
- verify before completion
- stop or request approval for private memory, credentials, persona, publishing, destructive operations, or self-evolution

This public skill does not include private persona or memory files.
