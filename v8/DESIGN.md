# V8 — Native Harness Control Plane

V8 keeps the mother Agent and native Codex loop as the default execution path. It adds small, independently switchable control-plane modules only when a task needs them.

- **Task Contract:** deterministically separates direct work, clarification, bounded recall and measured delegation candidates.
- **Context Economy:** injects at most 900 estimated tokens, removes duplicates and secrets, and measures evidence precision and utilization.
- **Evidence Trace V2:** stores allowlisted outcomes, verifier evidence and parent-child spans without raw prompts, private paths or hidden reasoning.
- **Deterministic control:** the mother Agent owns retries, escalation, hard budgets and acceptance; a child cannot rewrite these limits.
- **Harness Tax:** compares quality, token use, latency and correction cost against a native baseline.
- **Policy Lab:** policy changes stay in trial until distinct paired samples prove quality and a measurable benefit.
- **Skill Lifecycle V2:** repeated workflows move through candidate, shadow, replay and canary stages; critical failures revoke them.

Hooks remain disabled by default. V8 does not retain the previous semantic bridge or diary sidecar as active compatibility layers. Historical records remain in version history and archives, while the active runtime contains only the native-first control plane.
